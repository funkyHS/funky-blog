- 很多曾经接触过的技术，没有及时巩固，一段时间后，总是会忘记，当时的短时记忆没有转化为长期记忆，就已经被遗忘，在想回忆知识点的时候，如果没有留下学习记录，无从下手，所以在很早之前就开始写Blog，记录自己的学习过程
